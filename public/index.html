1

Download Data from Nash Portal

Get your delivery data from the Nash analytics portal:

  1. Go to https://portal.usenash.com/analytics
  2. Log in with your Nash credentials
  3. Select your date range (e.g., last week, last month)
  4. Click "Export to CSV" or "Download Report"
  5. Save the CSV file to your computer

Important: Make sure the CSV includes all required columns (Date, Store Id, Carrier, Total Orders, etc.). If columns are missing, contact Nash support.
2

Upload to Analytics Dashboard

Upload your Nash CSV file using the upload box above:

  1. Click the upload zone or drag and drop your CSV file
  2. Click "Upload File" button
  3. Wait for validation to complete (usually 5-10 seconds)
  4. Review the upload summary (total orders, stores, carriers)
  5. Click "Dashboard" in the navigation to view your analytics
üí°
Tip: Only California stores (2082, 2242, 5930, etc.) are analyzed. Non-CA stores are automatically filtered out and won't affect your metrics.
3

Add Spark CPD & Van OTD Data (Optional)

For enhanced reporting, add comparison metrics in the Admin page:

üìä Spark CPD (Cost Per Delivery)

  1. Navigate to Admin page (top navigation)
  2. Find the "Spark CPD Bulk Upload" section
  3. Create a CSV with columns: Store ID, Spark YTD CPD, Target Batch Size
  4. Example: 
    Store ID,Spark YTD CPD,Target Batch Size
5930,5.70,80
2082,5.50,80
2242,5.85,80
  5. Upload the file and verify success
  6. Dashboard will now show Van CPD vs Spark CPD comparison

üìà Van OTD % (On-Time Delivery)

  1. Navigate to Admin page
  2. Find the "Van OTD Bulk Upload" section (coming soon)
  3. Create a CSV with columns: Store ID, Van OTD %
  4. Example: 
    Store ID,Van OTD %
5930,95.5
2082,92.3
2242,98.1
  5. Upload the file and verify success
  6. Dashboard will display Van OTD % in KPIs
